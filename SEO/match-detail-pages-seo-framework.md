# Match Detail Pages SEO Framework

Date: 2026-05-25
Status: Applied to 104 match detail pages

## Core Keyword Rule

Each match detail page uses this page-level core keyword pattern:

```text
{Home Team} vs {Away Team} World Cup 2026 schedule
```

Example:

```text
South Korea vs Czechia World Cup 2026 schedule
```

## URL Rule

Each match page keeps the existing SEO-friendly URL:

```text
/world-cup-2026-match/{match-number}-{home-team-slug}-vs-{away-team-slug}/
```

Example:

```text
/world-cup-2026-match/2-south-korea-vs-czechia/
```

## Title Rule

Use the exact core keyword as the title:

```text
{Home Team} vs {Away Team} World Cup 2026 schedule
```

This keeps long team-name combinations under the 70-character target. The longest generated title after optimization is 59 characters.

## Description Rule

Use a concise description with the exact core keyword plus match facts:

```text
{Home Team} vs {Away Team} World Cup 2026 schedule: date, kickoff time, {Group/Stage}, {City} venue, timezone tools and team route links.
```

The longest generated description after optimization is 145 characters.

```text
H2: How to use this World Cup 2026 match schedule page
H2: {Home Team} vs {Away Team} related planning links
H2: {Home Team} vs {Away Team} FAQ
```

The H1 is visually hidden on the current match detail layout because the match center card is the visible first-screen experience, but the H1 remains in the HTML for SEO and accessibility.
