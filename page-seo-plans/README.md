# Page SEO Plans

This folder stores one-page SEO review and optimization records for each public page.

Purpose:

- Keep every page optimization traceable.
- Avoid batch template optimization.
- Record keyword targets, heading structure, content requirements, quality checks and follow-up work.
- Give future development a clear reference before editing any page.

Workflow:

1. Before optimizing a page, read `02-关键词方案.md`, `03-内容方案.md` and `04-页面SEO方案.md`.
2. Create or update the matching file in this folder.
3. Optimize one page at a time.
4. After optimization, update the page file with final metrics, heading structure, keyword placement and remaining issues.
5. Do not mark a page as complete unless it passes the page-level SEO checks.

File naming:

```text
page-seo-plans/world-cup-2026-schedule.md
page-seo-plans/world-cup-2026-schedule-pdf.md
page-seo-plans/world-cup-2026-schedule-excel.md
page-seo-plans/world-cup-2026-dates.md
```

Required fields for each page:

- Page URL
- Optimization status
- Reference model
- Competitor inspiration
- Page tool type
- Search intent
- Primary keyword
- Long-tail keywords
- Semantic keyword family
- SEO Title and Meta Description
- H1 / H2 / H3 structure
- Custom page blueprint
- Hero interaction
- Reusable action
- Retention mechanism
- Mobile behavior
- Content requirements
- Internal links
- External sources
- Schema requirements
- Current quality metrics
- Completed changes
- Remaining issues
- Next optimization action
