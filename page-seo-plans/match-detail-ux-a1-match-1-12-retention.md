# Phase Match Detail UX A.1

Date: 2026-05-25
Status: Completed
Scope: Match detail pages 1-12

## Goal

Improve early match detail pages so users can move from a single match into the most useful next planning path without returning to the full 104-match table.

## Applied Pages

```text
Match 1 through Match 12
```

## Added Module

The new `Early match planner` module appears below the top match center card.

It includes six retention paths:

```text
Confirm the kickoff window
Read the group route
Follow the teams
Plan the venue path
Keep watching the route
Save or share the schedule
```
